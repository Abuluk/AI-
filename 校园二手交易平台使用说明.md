# 校园二手交易平台使用说明

## 📱 平台简介

校园二手交易平台是一个基于AI技术的智能校园交易系统，为校园用户提供安全、便捷的二手物品交易服务。平台集成了多种AI智能推荐系统，包括AI大模型推荐、大数据推荐和AI增强推荐功能，为用户提供个性化的商品推荐体验。

### 🌟 核心特色
- **AI智能推荐** - 基于用户行为分析的个性化商品推荐
- **实时聊天系统** - 支持用户间私聊和系统消息通知
- **商家认证系统** - AI自动检测商贩行为，保障交易安全
- **社交功能** - 好友系统、黑名单管理、评论互动
- **移动端适配** - 响应式设计，完美支持手机端使用

## 🔐 用户认证

### 登录注册页面

#### 功能概述
提供用户注册和登录功能，支持用户名、邮箱、手机号等多种注册方式。

#### 主要功能
- **双模式切换** - 登录/注册模式一键切换
- **多种注册方式** - 支持用户名、邮箱、手机号注册
- **密码安全** - 密码强度检测和安全提示
- **记住登录** - 支持记住登录状态
- **忘记密码** - 提供密码找回功能

![登录注册页面截图](图片位置：登录注册页面.png)

#### 操作步骤

**注册新用户：**
1. 点击"注册"标签切换到注册模式
2. 填写用户名（必填）
3. 选择注册方式：邮箱或手机号
4. 输入对应的邮箱地址或手机号码
5. 设置登录密码
6. 确认密码
7. 点击"创建账户"完成注册

**用户登录：**
1. 点击"登录"标签切换到登录模式
2. 输入用户名
3. 输入密码
4. 可选择"记住我"保持登录状态
5. 点击"登录"进入平台

![登录流程截图](图片位置：登录流程.png)

#### 密码找回
1. 在登录页面点击"忘记密码？"
2. 输入注册时使用的邮箱或手机号
3. 系统发送验证码
4. 输入验证码并设置新密码

![密码找回截图](图片位置：密码找回.png)

## 🛍️ 商品管理

### 商品发布页面

#### 功能概述
用户可以在此页面发布新的二手商品，系统提供AI智能辅助功能帮助完善商品信息。

#### 主要功能
- **AI智能填充** - 根据上传图片自动生成商品标题和描述
- **多图片上传** - 支持上传多张商品图片
- **分类选择** - 提供详细的商品分类选项
- **价格设置** - 支持精确到分的价格设置
- **商品状态** - 选择商品新旧程度
- **位置信息** - 设置商品所在位置

![商品发布页面截图](图片位置：商品发布页面.png)

#### 操作步骤
1. **填写基本信息**
   - 输入商品标题（AI可自动生成）
   - 填写商品描述（AI可自动生成）
   - 设置商品价格

2. **选择分类和状态**
   - 从下拉菜单选择商品分类
   - 选择商品新旧程度

3. **上传商品图片**
   - 点击上传区域选择图片

4. **设置位置信息**
   - 输入商品所在位置
   - 选择是否显示具体位置

5. **发布商品**
   - 检查所有信息无误
